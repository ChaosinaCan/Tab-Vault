Tab Vault Help

Qu'est-ce que Tab Vault?

Tab Vault est un endroit où vous pouvez laisser les onglets que vous souhaiter retenir sans les laisser chargés en mémoire ainsi que sur votre barre d'onglets. Tout comme la barre d'onglets d'Opéra, vous pouvez réorganiser et grouper les onglets, de même que restaurer des onglets supprimés grâce à la corbeille de Tab Vault. Tab Vault affiche aussi le nombre d'onglets que vous avez enregistrés pour vous rappeler d'y retourner.

Comment utiliser Tab Vault?

Cliquez sur le bonton de l'extension pour ouvrir Tab Vault. Tab Vault affiche le nombre d'onglets qu'il stocke sur ce bouton. Cette option (et beaucoup d'autres) peut être modifiée à partir de la page de configuration.

L'interface

Tous vos onglets enregistrés se retrouvent ici. Les onglets supprimés sont quant à eux accessibles en cliquant sur le bouton Corbeille.

Enregistrer des onglets

Cliquez sur Enregistrer l'Onglet pour enregistrer l'onglet courant. Cliquez avec le bouton droit de la souris sur le bouton Enregistrer l'Onglet pour ouvrir un menu contextuel avec davantage d'options. Il est aussi possible de cliquer avec le bouton droit sur un groupe d'onglets et de choisir Enregistrer l'Onglet Courant Ici pour enregistrer un onglet directement dans un groupe.

La Liste d'Onglets

Faites glissez les onglets pour les réorganiser. Pour supprimer un onglet, cliquez sur le bouton X situé à côté de chaque onglet ou déposez-le sur le bouton Corbeille en haut du menu.

Pour ouvrir un onglet, cliquez dessus. Pour ouvrir un onglet en arriere-plan, cliquez dessus avecc le bouton du milieu ou laissez la touche Ctrl appuyée en cliquant. Pour ouvrir un onglet dans la page courant, faites-le glisser du menu jusqu'à la page web ouverte. Vous pouvez aussi cliquez avec le bouton droit sur un onglet pour afficher d'autres options.

Groupes d'onglets

L'implementation des groupes d'onglets de Tab Vault est conçu pour être très similaire à la fonction d'empilement d'onglets d'Opera. Ils peuvent contenir un nombre illimité d'onglets et disparaissent lorsqu'ils sont vides.

Pour créer un groupe, faites glisser un onglet jusqu'à ce que votre curseur soit au-dessus de la flèche apparaissant à droite de de l'onglet. Ensuite, lorsque l'onglet foncit, relâchez le tout pour grouper les onglets.

Cliquez sur le titre d'un groupe d'onglet pour afficher ou masquer les onglets qu'il contient. Pour modifier le titre d'un groupe, il suffit de cliquer sur l'icône du stylo située à droite du titre.

Les groupes d'onglets peuvent être déplacés par glisser-déposer, de la même façon qu'un onglet unique. Il est aussi possible d'inclure ou d'exclure des onglets d'un groupe par cette méthode. Lorsque tous les onglets d'un groupes sont supprimés ou déplacés vers l'extérieur, le groupe disparaît. De plus, faire glisser un groupe vers la Corbeille le supprimera avec tout son contenu.

La Corbeille Restaurer des onglets supprimés

Lorsque vous supprimez un onglet, c'est ici qu'il se retrouve. Cliquez sur un onglet de cette liste pour le restituer à la liste d'onglets. Cliquez sur Vider la liste d'onglets fermés pour supprimer, de manière permanente, tous les onglets de la corbeille.

Fonction supplémentaires

Sessions

Tab Vault peut importer et exporter des fichiers de session et de signets à partir de la section Importer/Exporter de la page de configuration.

Pour exporter vos onglets Tab Vault en tant que fichier de session, cliquez sur Exporter en tant que session. Opera ouvrira un nouvel onglet. Choisissez Menu > Page > Enregistrer sous... (ou Ctrl+S) pour enregistrer cet onglet comme fichier .win. Il est aussi possible d'exporter comme fichier de signets d'Opera, grâce au bouton Exporter .adr.

Pour importer un fichier de session ou de signets, utilisez le selécteur de fichier pour naviguer jusqu'au fichier souhaité (.win ou .adr), puis cliquez sur "Importer Session".

Pour localiser l'endroit où Opera conserve les fichiers de session, aller à opera:about. Le chemin est indiqué à coté de "Session enregistrée". Si vous désirez que vos sessions exportées apparaissent dans la liste de sessions d'Opéra, enregistrez-les à cet endroit.

Selon certains rapports que j'ai reçus, un problème existe avec Opera, ce qui pourrait faire en sorte que toutes les données enregistrées par TabVault soient perdues. Bien que ce phénomène se produise très rarement, il est recommendé d'exporter régulièrement vos onglets comme sauvegarde, juste au cas où un problème surviendrait.

Boutons et Raccourcis

Tab Vault expose des fonctions JavaScript afin de lui permettre d'être controlé à partir de pages web ou de sections du programme Opera. Cela signifie qu'il est possible d'associer un raccourci de clavier (ou bouton) pour rapidement enregistrer l'onglet en cours.

Pour activer cette fonctionnalité, allez dans la section Boutons et Raccourcis de la page de configuration et créez un mot de passe. Ce mot de passe empêchera des sites malfaisants d'enregistrer du contenu dans Tab Vault. Il n'est pas nécessaire que vous l'appreniez par coeur, vous pouvez donc cliquer sur le bouton "Générer" pour créer un mot de passe aléatoire. Ce mot de passe ne sécurise pas le contenu de Tab Vault. Il sert seulement à empêcher les sites web d'intéragir avec le contenu de Tab Vault.

Pour créer un bouton permettant d'enregistrer l'onglet en cours, il suffit de cliquer le bouton avec l'icône désirée. Il sera alors ajouté dans la section "Mes boutons" de la fenêtre d'apparence, à partir de laquelle il est possible de le glisser-déposer sur n'importe quelle barre d'outils. (Un bug d'Opera fait présentement en sorte que ce bouton apparaisse plus gros que normal, il est donc recommandé d'utiliser un des petits boutons.)

Pour créer un raccourci de clavier ou de souris, copiez le texte affiché sous "Enregistrer action d'onglet", puis allez à Menu > Paramètres > Préférences > Avancé > Raccourcis, modifier votre configuration de clavier ou souris, et créez un nouveau raccourcis en utilisant le texte précédemment copié en tant qu'action.

Pour enregistrer un onglet à partir de JavaScript, utilisez la fonction window.opera.tabvault.save(pass, url, title, icon) où "pass" est votre mot de passe, "url" est l'URL de l'onglet, "title" est le titre de l'onglet et "icon" est l'URL de l'icône de l'onglet. Ne mettez aucun paramètre, outre le mot de passe, pour enregistrer l'onglet en cours.

Note: Un bug d'Opera empêche présentement Tab Vault de détercter l'onglet d'un onglet s'il a été enregistré à l'aide d'un bouton ou raccourci.
