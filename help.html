Tab Vault Help

What is Tab Vault?

Tab Vault is a place to store the tabs you want to get to later without leaving them loaded in memory and on your tab bar. Just like with Opera's tab bar, you can reorder and group tabs and restore deleted tabs from Tab Vault's trash list. Tab Vault also shows you the number of tabs you have stored to remind you to get back to them.

How do I use Tab Vault?

Click the extension button to open Tab Vault. The button also shows how many tabs are stored. You can customize this and many other things from the options page.

The Interface

All your saved tabs are kept here. Deleted tabs can be accessed by clicking the Trash button.

Saving Tabs

Click Save Tab to save the current tab. Right-click the Save Tab button to open a context menu with more options. You can also right-click a tab group and choose Save Current Tab Here to save a tab directly to a group.

The Tabs List

Click and drag tabs to reorder them. To delete a tab, click the X button next to it or drag it to the Trash button at the top.

To open a tab, click it. To open a tab in the background, middle click it or hold Ctrl while clicking it. To open a tab in the current page, drag it from the pop-up to the web page. You can also right-click a tab for more options.

Tab Groups

Tab groups work much like they do on Opera's tab bar. They can hold any number of tabs and will disappear if they become empty.

To create a group, start dragging a tab and move the cursor to the right so that the tab's border darkens. Next, drag the tab over another tab so that the other tab darkens. Finally, drop the tab and the two tabs will be grouped.

To expand or collapse a group, click its title. You can also rename a group by hovering your mouse over the group, then clicking the pen icon on the right.

Tab groups can be dragged and moved just like tabs. You can drag tabs in and out of expanded groups. Grouping a group with another group will merge the groups. The group will disappear if all of its tabs are removed. You can also delete an entire group at once by dragging it to the Trash button at the top.

The Trash List

When you delete a tab, it goes to the trash list. Click on a trashed tab to restore it to the tabs list. Click Clear List of Closed Tabs to permanently delete all trashed tabs.

Extra Features

Sessions

Tab Vault can import and export Opera session and bookmarks files from the Import/Export section of the options page.

To export your vault's tabs as a session file, click Export Session. Opera will open a new tab. Use Menu > Page > Save As... (or Ctrl+S) to save this tab as a .win file. You can also export as an Opera bookmarks file with the Export .adr button.

To import a session or bookmarks file, use the file picker to select the .win or .adr file, then click "Import Session".

You can find where Opera keeps your session files by going to opera:about and reading next to "Saved session". If you want your exported sessions to appear in Opera's session list, save them to this location.

I have received reports where a problem with Opera caused it to lose all of Tab Vault's saved data. While this is very rare, I recommend you periodically export your tabs for safekeeping, just in case something does go wrong.

Buttons and Shortcuts

Tab Vault exposes JavaScript functions so that it can be controlled from web pages or other parts of Opera. This means you can set up a keyboard shortcut or a button to quickly save the current tab.

To enable this, go to the Buttons and Shortcuts section of the options page and set a password. This password keeps malicious sites from saving things to your vault. You don't have to remember it, so just click the "Generate" button to create a random password. This password does not secure the vault. It only prevents web sites from interacting with the vault.

To make a button to save the current tab, just click the button with the icon you want. It will be added to the "My Buttons" section of the appearance window, from which you can drag it to any toolbar. (A bug in Opera currently makes this button bigger than normal, so use one of the smaller buttons.)

To make a keyboard shortcut or mouse gesture, copy the text shown under "Save tab action", then go to Menu > Settings > Preferences > Advanced > Shortcuts, edit your mouse or keyboard setup, and create a new shortcut using the copied text as the action.

To save a tab from JavaScript, use the function window.opera.tabvault.save(pass, url, title, icon) where "pass" is your password, "url" is the URL of the tab, "title" is the tab's title, and "icon" is the URL of the tab's favicon. Omit everything after "pass" to save the current tab.

Note: A bug in Opera currently prevents Tab Vault from detecting a tab's favicon when saved with a button or shortcut.
