Tab Vault Help

By Joel Spadin

Neu bei Tab Vault?

Wenn Sie nicht sicher sind, was Tab Vault ist oder wie man es verwendet, so lesen sie einfach weiter. Wenn sie ein Handbuch nur als letzten Ausweg zu Rate ziehen, so könnten Sie folgende Links interessieren:

Was ist Tab Vault?

Tab Vault kann verwendet werden um die Tabs innerhalb von Opera zwischenzuspeichern. Der Vorteil daran: die Tabs verbrauchen keinen Arbeitsspeicher und keinen Platz in der Tableiste, sind aber trotzdem einfach und schnell zu erreichen. Genau wie in der Leiste können Sie Tabs neu anordnen und gruppieren (Stacking) sowie gelöschte Tabs aus dem Papierkorb wiederherstellen; außerdem können diese als eine Operasitzung gespeichert werden. Weiterhin wird die Anzahl der von Ihnen gespeicherten Tabs als Erinnerung im Toolbar-Symbol angezeigt.

Wie benutze ich Tab Vault?

Klicken Sie auf das Toolbar-Symbol der Erweiterung um Tab Vault zu öffnen. Dieses zeigt hier die Anzahl der gespeicherten Tabs an. Das und viele andere Dinge können sie unter den Einstellungen ändern.

Die Tab Vault GUIDie Schnittstelle zum Benutzer

Tab Vault hat zwei Listen: Tabs und Papierkorb. In "Tabs" befinden sich die gespeicherten Tabs; in "Papierkorb" werden die gelöschten Tabs zwischengespeichert, damit diese bei Bedarf wierhergestellt werden können. Klicken sie oben am Popup auf Tabs bzw. Papierkorb (Trash), um zwischen den beiden Listen zu navigieren. (Anmerkung: Ein leerer Papierkorb ist nicht aktivierbar)

Die Tabliste Tabs öffnen und verwalten

Klicken Sie Tab speichern (Save Tab) um den aktuell geöffneten Tab zur Tabliste hinzuzufügen. Klicken und ziehen sie die Tabs, um deren Anordnung zu ändern. Klicken sie auf das X eines Tabs um diesen in den Papierkorb zu verschieben. Dies geht auch, indem sie den Tab auf dem Papierkorblink ablegen. In der Tabliste sieht man nur die Seitentitel; beim Überfahren mit der Maus werden aber zusätzlich die URLs angezeigt. Dies kann man in den Einstellungen auch anpassen. .

Es gibt drei Wege einen Tab zu öffnen.

  1. Einfach darauf klicken, um ihn in einem neuen Tab zu öffnen.
  2. Strg halten und auf den Tab klicken um diesen im Hintergrund zu öffnen.
  3. Einen Tab aus der Liste ziehen und in der aktuellen Webseite ablegen, um diesen hier zu öffnen.

Der Papierkorb Gelöschte Tabs wiederherstellen

Wird ein Tab gelöscht landet er hier. Klicken Sie auf einen gelöschten Tab um ihn wiederherzustellen. Klicken sie Papierkorb leeren (Clear list of closed tabs) um alle gelöschten Tabs endgültig zu entfernen. Anmerkung:Im Papierkorb können die Tabs nicht neu angeordnet werden (wofür auch? :)).

TabgruppenSchaffen wir Platz

Tab Vault unterstützt seit Version 2.0 auch Tabgruppen. Diese haben einen Erweiterungspfeil anstatt eines Favicons, außerdem gibt es kein X-Symbol. Statt dessen erscheint an dieser Stelle ein Stiftsymbol, wenn man mit der Maus darüber fährt (dazu später). Um mit zwei Tabs eine Gruppe zu erstellen gehen Sie wie folgt vor:

  • Klicken und halten sie den Tab, mit welchem Sie eine Gruppe eröffnen wollen
  • bewegen Sie den Mauszeiger solange nach rechts auf diesem Tab, bis der Pfeil ganz rechts nach unten zeigt (siehe Abbild unten; der entsprechende Bereich ist mit einem roten Kasten gekennzeichnet)
  • nun ziehen Sie diesen Tab solange über den anderen Tab, bis letzterer sich dunkel verfärbt.
  • Als letztes lassen Sie den Tab hier fallen um diese beiden zu einer Gruppe zu vereinigen.

Klicken sie auf den Namen einer Tabgruppe, um diese zu erweitern bzw. wieder einzuklappen. Um einen Gruppennamen zu ändern klicken sie auf das Stiftsymbol, welches beim Überfahren des Gruppentabs erscheint. Tabgruppen kann man ziehen und ablegen wie normale Tabs auch. Weiterhin können in diese wenn sie erweitert wurden neue Tabs hineingezogen werden, um sie zur Gruppe hinzuzufügen, sowie bestehende Tabs herausgezogen werden, um diese aus der Gruppe zu entfernen. Eine Tabgruppe verschwindet bzw. verwandelt sich in einen einzelnen Tab, wenn sie nur noch einen Eintrag enthält. Eine komplette Gruppe kann man löschen indem man diese in den Papierkorb verschiebt.

Die Implementation der Tabgruppen ist hier so gut wie identisch mit denen von Operas Tab Stacking. Wenn man eine Tabgruppe auf einen Tab fallen lässt - so wie man eine Tabgruppe erstellen würde - so wird daraus eine neue Gruppe erstellt mit dem neuen Tab als ersten Eintrag. Wird eine Gruppe auf eine andere fallen gelassen, so wird erstere in letztere aufgenommen. Lässt man einen Tab auf eine eingeklappte Tabgruppe fallen so wird er dieser hinzugefügt.

Zusätzliche Features

SitzungenTablisten speichern und wieder herstellen

Tab Vault kann Opera-Sitzungsdateien exportieren und importieren; dies geschieht über den Import/Export-Abschnitt der Einstellungsseite.

Zum Exportieren Ihrer Tabs aus Tab Vault klicken sie "Sitzung exportieren"; Opera wird dann einen neuen Tab öffnen. Über Menü > Seite > Speichern..., erreichbar über den Hotkey Strg + S, kann man diese Seiten dann als .win-Datei speichern.

Um eine Sitzung zu importieren öffnen Sie als erstes die .win-Datei in einem Texteditor, kopieren den kompletten Inhalt in die Textbox auf der Einstellungsseite und klicken dann auf "Sitzung importieren".

Tipp: Wenn man eine exportierte Sitzung im selben Verzeichnis ablegt in dem Opera seine eigenen Sitzungen speichert, so wird sie danach im Sitzungsmenü von Opera angezeigt. Um zu zu ermitteln, wo man die Datei speichern muss, öffnet man opera:about und schaut sich den Eintrag hinter "Gespeicherte Sitzungen" an; dieser zeigt auf die automatisch gespeicherte Sitzungsdatei. In das gleiche Verzeichnis kommt dann die exportierte Sitzung.

Buttons und Hotkeys

Tab Vault bietet JavaScript-Funktionen, über welche das Programm von Webseiten oder von anderen Operaprogrammen gesteuert werden kann. Das bedeutet, dass Sie einen Hotkey, eine Mausgeste und/oder einen Button erstellen können, um einen Tab schnell zu speichern.

Um das zu aktivieren gehen Sie zur "Buttons und Shortcuts" - Sektion der Einstellungen und setzen sie ein Passwort. Dieses verhindert, dass bösartige Websites etwas in Ihrem TabVault speichern. Sie brauchen sich das Passwort nicht zu merken, daher können Sie hier einfach "Passwort erstellen" klicken um ein zufälliges Passwort zu erstellen.

Um einen Button für das Speichern des aktuellen Tabs zu erstellen wäheln Sie unter Buttons zum Speichern eines Tabs einen aus und klicken darauf. Unter Menü > Erscheinungsbild > Schaltflächen > Eigene erscheint dieser dann und kann von hieraus wie jeder anderen Button auch in irgendeine Werkzeugleiste geogen werden. (Durch einen Fehler in Opera erscheint der Button zur Zeit größer als normal, verwenden Sie daher einen der kleineren)

Um einen Hotkey oder eine Mausgeste zu erstellen kopieren Sie zuerst den Text, welcher unter "Das wird zum Speichern eines Tabs ausgeführt" steht. Danach öffnen Sie Menü > Einstellungen > Einstellungen > Erweitert > Schnellzugriff, bearbeiten das entsprechende Setup für Mausgesten bzw. Tastenkombinationen, und erstellen einen neuen Hotkey mit dem kopierten Text als Aktion.

Um einen Tab aus JavaScript heraus zu speichern benutzen Sie die Funktion window.opera.tabvault.save(pass, url, title, icon), wobei "pass" Ihr (wahrscheinlich gerade erst generiertes) Passwort ist, "url" die URL des Tab, "title" der Name des Tabs, und "icon" die URL zum Favicon des Tabs. Lassen Sie alles nach dem Passwort weg, sow ird der aktuelle Ta gespeichert.

Anmerkung: aufgrund eines Bugs in Opera ist es TabVault nicht möglich das Favion eines Tabs zu ermitteln, wenn dieses mittels eines Buttons oder Hokeys gespeichert wurde.

Falls Fehler auftreten sollten

Das Design der Erweiterung erschwert es zwar, das Bugs einen schwerwiegenden Datenverlust verursachen, trotzdem habe ich (sehr selten) Rückmeldungen bekommen von Leuten, deren Tabs komplett aus der Tab Vault verschwunden sind. Das kann passieren wenn Opera die Daten der Erweiterung verliert, sei es durch einen Absturz des Browsers, einen noch nicht entdeckten Bug in Tab Vault, oder etwas komplett anderes. So oder so, ich empfehle dringend, sicherheitshalber regelmäßig Ihre Tabs zu exportieren, nur für den Fall das etwas passiert.
